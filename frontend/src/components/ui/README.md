# UI Components Directory (shadcn/ui)

This directory contains standard primitive UI components, usually generated via shadcn/ui.
Examples:
- Button
- Input
- Dialog
- DropdownMenu
- Card

## Guidelines
- Avoid customizing files directly unless it represents a global theme change for that component.
- Use tailwind classes to customize styling.
- All files in this folder should remain atomic and fully reusable.
